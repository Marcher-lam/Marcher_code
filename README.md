## ℹ️ 关于

本仓库旨在整理和分享人工智能相关的理论与实践项目，包括：

- **数学基础**：线性代数、概率论、优化、微积分等  
- **机器学习 (ML)**：监督学习、无监督学习、特征工程等  
- **深度学习 (DL)**：神经网络、卷积网络、循环网络、Transformer 等  
- **强化学习 (RL)**：策略优化、价值函数、Actor-Critic 方法等  
- **大型语言模型 (LLM)**：GPT、BERT、LLM 微调与应用  
- **视觉语言模型 (VLM)**：多模态模型，如 CLIP、BLIP 等  
- **视觉语言导航 (VLN)**：视觉语言导航任务与算法  
- **视觉语言代理 (VLA)**：多模态智能体、任务执行  
- **智能体 (Agent)**：强化学习与多智能体系统实践
